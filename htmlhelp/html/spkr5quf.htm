Vented dialog (Calculate Vented for Enclosures)

Use this command to select an optimum ported box enclosure for a specific driver.

Fields:    
  Alignment Select an alignment for the enclosure. Changing alignments changes the Box Volume and Port Length fields, as well. Chebychev (C4) is probably the best workhorse of the bunch. QB3 only works for driver Q’s less than 0.4.
  Series Resistance Include cable and crossover series resistance in this amount. Changing this will also change Box Volume and Port Length.
  Number of Ports The number of ports you are putting the enclosure.
  Number of Drivers The number of drivers in the enclosure.
  Port Diameter The inside diameter of each port. If using a rectangular port, use the diameter which produces the same internal area.
  Port Length The necessary length for the alignment. Changing this also changes box volume to keep tuning frequency constant.
  Box Volume The necessary internal box volume for the desired alignment.
  Tuning Frequency The box tuning frequency. If you change this both Port Length and Box Volume will change to keep the “box Q” relatively constant.
Commands    
  Compare Press this command button to compare the box based on the dialog entries to the current box. See below.

Note that Box Volume and Port Length can be set independently of the alignment. Changing either parameter will alter the box from a standard alignment, but can still attain a good frequency response.

This command requires that a valid driver be entered into the enclosure (see Enclosure Properties) and the driver has valid T/S parameters. Get Speaker Workshop to estimate the driver T/S parameters (via impedance measurements) or enter them manually prior to executing this command.

This command changes the current properties of the enclosure, to match the settings selected in the Vented dialog.

The Compare Command

When you press the Compare button, Speaker Workshop builds an enclosure named “Compare” in the System directory, generates the desired curves for the compare enclosure (based on the menu settings) and charts the Compare frequency response vs the current frequency response on-screen.

Note that the Compare enclosure and associated datasets are not deleted when you exit this dialog! This allows you to create comparison charts for other than frequency response (using the Compare.xxx datasets and the current datasets) and then using the Vented dialog you can compare impedance, cone excursion, and group delay.

For example, after running the Vented dialog once, open the myenclosure.Group Delay dataset, Add the System\Compare.Group Delay dataset and then bring up the Vented dialog again (while keeping the group delay dataset visible on screen). As you press compare you will note that the secondary chart (group delay) also changes each time the command is executed.
