Smooth dialog (Transform menu for Datasets)

Use this dialog to smooth data using an averaging process.

Fields    
  Smoothing Range Select a range to smooth over.
  Averaging Select arithmetic, RMS, or absolute (amplitude only) averaging.

Note that the averaging is done on a continuous basis. The averaged data will not be a bunch of horizontal lines, but each data point is the averaged result.

The program supports three types of averaging:

Arithmetic The values are added together arithmetically using the vector amplitude and phase values.
Absolute The amplitude and phase values are smoothed independently.
RMS The vector values are added together using root-mean-square arithmetic.

This command is most commonly used to convert ungated acoustic data into 1/3 octave average room data.

This command changes the current dataset.
