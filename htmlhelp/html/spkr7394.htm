Merge Port Response dialog (Calculate Merge Port Response for Enclosures)

Use this command to correctly merge a nearfield woofer and port response to get the combined woofer nearfield response of a ported enclosure.

Fields:    
  Driver Nearfield Select the appropriate nearfield driver response. Press the ? to select from the resource tree. This will be filled in with the nearfield response for the driver in the enclosure – by default.
  Port Nearfield Select the appropriate nearfield port response. Press the ? to select from the resource tree.
  Piston Area This shows the piston area for the driver in the enclosure. This is not editable. To change it go to the driver properties and change it there.
  Port Area This shows the area of the port in the enclosure. This is not editable. To change it go to the Enclosure Properties and change it there.

When this command completes it will build a new dataset named Myenclosure.Merged Response that contains the appropriate summed response.
