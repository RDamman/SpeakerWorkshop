Editing networks with a mouse

Networks are graphical things. Although Speaker Workshop lets you edit a network entirely via numbers (primarily in the network properties dialog), you can also do a lot of things with the mouse. To wit

Component Editing Mouse Commands

Press (not on a component) and drag, then release inside the view area. This will draw a rubber-band rectangle. When you release, the components within the box are selected.
Right click This brings up a menu of standard operations.
Click an object This highlights the object and enables Delete and node connection.
Double-click an object This brings up the network properties dialog, with the clicked object ready for editing.
Press and drag an object or a collection of objects The selected objects move through the view area. Note that all node connections are retained during this move. Lines are drawn automatically.

Connection Mouse Commands

To edit component connections with the mouse, you start by selecting a component. This makes its nodes look like (two) large empty circles. When your mouse enters a node you will note that the cursor changes shape to show you can connect the nodes.

Click a node and drag the line to another node This connects the two nodes, dropping any other connections between those two nodes.
Shift key w/above Press and hold shift while releasing the mouse button to retain all other connections.
Drop the line anywhere This cancels the connection operation.
Click a node This is equivalent to dragging a node to itself and it removes all connections to that node.
