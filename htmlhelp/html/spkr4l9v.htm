Intermodulation Distortion command (Measure menu, for Signals)

Use this command to measure the intermodulation distortion of a circuit or a speaker. This command generates different output signals depending on which distortion option is selected in the Intermodulation DistortionHIDD_MSRINTERMOD dialog.

This command creates or replaces a dataset named mysignal.Intermod. It may also change a number of datasets in the System folder.
