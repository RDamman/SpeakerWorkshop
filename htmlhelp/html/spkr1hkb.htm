Waterfall Dialog (Calculate Waterfall for Datasets)

Waterfall plots can only be calculated on Time datasets.

Fields    
  Source The source time data to be analyzed. By default this is the current dataset.
  Reference (optional) The reference time data. This should be the actual reference data captured during the pulse capture.
  Windowing The FFT windowing applied during the analysis. Default is Uniform (none).
  Sample Time The amount of time to calculate the waterfall for. This is the decay time you want to examine (often 1-5ms).
  Number of Samples How many different plots to generate for the waterfall. They will be spaced sample time / number of samples apart.
  Smoothing You can apply smoothing in pieces of an octave (e.g. .3 = 1/3 of an octave). Usually 1/10 of an octave (0.1) is plenty.
  Sample Size (Time Range) Select the range of data to be analyzed You can choose to use the markers or set the range manually..

Actions    
  ? Press the ? button to browse for a reference or source response.
  X Press X to clear the reference response.
  OK Click OK to perform the calculation. This creates many new datasets (one per plot line), a chart and a folder to contain the datasets.

Click OK to calculate the waterfall.
