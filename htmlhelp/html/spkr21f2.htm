Impedance command (Calculate menu for Datasets)

Use this command to calculate an impedance dataset for a set of frequency data. This command performs the same calculations as Measure Impedance, but after the fact for Post Mortem analysis. It uses two datasets (input data and reference data) and any channel calibration data to calculate a final response.

Impedance Jig settings such as reference resistor are found in the Option Preferences Dialog

This command creates a new dataset with a Z suffix.

For more information see the Impedance Dialog.
