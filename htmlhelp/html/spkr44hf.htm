Frequency Response Dialog (Calculate Frequency Response for Datasets)

Use this dialog to perform the same calculations as Measure Frequency Response, but after the fact for Post Mortem analysis. It can use two datasets (input data and reference data) and any channel calibration data to calculate a final response.

Fields    
  Windowing The FFT windowing applied during the analysis. Measurements always use Uniform (none).
  Calibration Select no calibration to perform a transfer function analysis. Use channel calibration to correct for L-R imbalances in the test sound board. Use microphone calibration when the input data is acoustic.
  Reference response Equivalent to the second channel during measurements. The final response is Input / Reference.
  Time Range Select the range of data to be analyzed.

Actions    
  ? Press the ? button to browse for a reference response.
  OK Click OK to perform the calculation. This creates a new dataset with a Freq suffix.
