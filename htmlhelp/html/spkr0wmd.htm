The Project Tree

The Project Tree is the window on the left of the main display that contains the tree diagram showing all resources in the project. Folders provide places to put resources (just like folders in Win95). The tree has a number of mouse shortcuts:

Double Click Open the clicked resource
Click and drag Move a resource from one folder to another
Shift Click and drag Copy a resource from one spot to another
Right click Bring up a context popup menu with tree operations

The project tree can be hidden / shown by selecting the View Project Tree command.

The project tree can be picked up and floated or attached to the right edge of the program window as well as attached to the left edge (default) by grabbing the project tree window outside of the tree display.

The tree can also be seen in a view window (just like the chart view). The default view is just such a tree. This view may be preferable to the always-on-top Project Tree.

The icons to the left of each resource name describe the resource type. They are:

Charts

Datasets

Response vs Frequency

Impedance vs Frequency

Response vs Time

Drivers

Enclosures

Folders

Networks

Signals
