FFT dialog (Calculate FFT for Datasets)

Select the time range and FFT windowing with this dialog.

Fields    
  Windowing Windowing is a mechanism for reducing the edge effects of non-repeating data. Select the windowing desired.
  Time Range Select a time range to be transformed.

Windowing options are:

Bartlett  
Blackman  
Hamming  
Hann  
Kaiser  
Parzen  
Pretel  
Triangular  
Uniform No windowing is applied.
Welch  

Click OK to perform the FFT transform. This creates a new dataset with an Fft suffix.
