Burst tab (Signal Properties dialog)

This page contains settings for the Burst type of signal. Fields are:

Frequency Start The starting frequency for the sweep.
  End The ending frequency for the sweep.
Cycles   The number of cycles to generate the burst for (this is based on the starting frequency).

A burst signal is a shaped tone burst. The burst signal is a sweep from start frequency to end frequency shaped by a half cycle of a sine wave that lasts just long enough for the desired number of cycles.
