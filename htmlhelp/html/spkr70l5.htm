Set Equivalence command (Driver menu for Drivers)

This command brings up the Set Equivalence dialog, which lets you manually set the driver equivalent elements.

If you are unhappy with the automatic estimation you can use this dialog to reset the parameters manually. The dialog box contains edit fields for each of the driver equivalent circuit elements (Q’s change as you change the fields) along with a button to let you chart the impedance of the new circuit.

This command changes the current properties of the driver, inserting new T/S parameters based on input data.
