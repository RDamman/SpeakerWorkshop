Rename dialog (Resource Rename)

Use this dialog to rename a resource.

Fields:    
  Name Enter the new name of the resource.

Click OK to rename the resource. Any references to the resource are automatically updated to the new name. This will not move the resource to a different folder.

This is often used to rename a result dataset (such as myinput.Fft or mydriver.FreeAir) to allow comparison with a new calculation. Calculations store their results in a dataset by name, not reference.
