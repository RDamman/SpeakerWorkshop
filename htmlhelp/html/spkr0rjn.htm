Calibrate Dialog (Options Calibrate)

Use this dialog to select (or create) calibration curves for the system. There are two calibration curves: the microphone calibration and the channel calibration. Both are optional, although a microphone will always require a calibration for accurate results.

Fields:

Channel Difference The amount that left and right differ (by frequency in dB)
Microphone Response The difference between the mic response and flat (by frequency in dB).
Amplifier Reference Response The pulse response of the amplifier + sound card combination (by time).

Actions:

Test Click the Test button to have the response tested.
None Click the None button to use no calibration curve.
Browse Select from the set of available response curves.

Results:

The calibration curves selected or created are saved when the current document is saved. Calibrations are linked to the current document because calibration curves are members of the document.

Clicking a test button executes a calibration wizard. This simple step by step process walks through the test.

Comment:

Although you can cancel this dialog and not accept changes, test measurements that occur in the system folder are not undoable. Thus, if you use the default channel calibration (System\Measurement.Calib) you can not really cancel the measurement once it has taken place. If you want to be able to cancel, make a copy of the calibration before proceeding through a test cycle.

Note:

In order to get a valid microphone response chart you will need to import one provided by the microphone manufacturer. See the section on import data formats.
