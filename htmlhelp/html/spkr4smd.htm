Memory Requirements

Speaker Workshop can use up plenty of memory. This memory is taken from the Windows 95 virtual memory manager and need not be real RAM, but it must be available.

To calculate approximate memory requirements for a project….

Document Each document that is loaded is stored entirely in memory and will require storage as below.
Undo/Redo Buffer When an operation proceeds, the undo/redo buffer obviously contains the prior dataset. For operations like transforms, that can be a pretty big object (see below)

To clear the undo/redo buffer, just save the document (via File Save). This can potentially free up large amounts of memory.

Memory requirements per data point

When an object is a dataset, it occupies approximately 8 times its length. Thus, a 16K measurement will take up 8*16K = 128K Bytes of virtual memory. Put it another way, every 8 datasets take up 1MB. If you instead use a 128K measurement the storage requirement balloons to 1MB per dataset!
