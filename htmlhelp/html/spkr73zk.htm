Pulse Response command (Measure menu for Drivers)

Use this command to measure the pulse response of the driver in the enclosure.

The measurement is performed by sending an MLS (Maximum Length Sequence) wave out through the sound board. The resulting source input wave becomes the pulse response. This measurement is not gated. It uses only one channel of input, as well.

Change the volume, the length of the sequence, and other settings in the Options Preferences dialog.

This command is strictly optional and mainly for post mortem (long after the recording) analysis of data.

This command changes the current properties of the driver, inserting a new dataset entry into one slot.
