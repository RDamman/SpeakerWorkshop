Check Sound Card more Input information
(Options Wizard Check Sound Card)

This dialog contains more technical information about your input capabilities. Press Next and Prev to check out other boards (if you have more than one board installed).

Fields are:

Manufacturer ID Ignore this field.
Product ID Ignore this field
Channels Stereo or Mono
Driver Version The version of the hardware driver you are running.
Name The external name of the driver.
Formats Formats supported on input: mono/stereo, number of bits, and sample rate.
