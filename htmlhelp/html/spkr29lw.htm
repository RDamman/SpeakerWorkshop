Axis tab (Chart Properties dialog)

This page shows up as X Axis, Y Axis, and Y2 Axis. The X-axis is the horizontal (often frequency) axis. The Y-axis is usually the amplitude display. The Y2 axis is usually the phase display. For each axis, you can set:

Title The axis title is printed next to the axis scale. If this field is empty there is no axis title.

Scale controls are:

Logarithmic Use a logarithmic scale or a linear scale.
Auto minmax If this is selected, then the scale range is determined automatically from the data.
Maximum If manual scaling, this is the maximum data value plotted.
Minimum If manual scaling, this is the minimum data value plotted.

The axis style is defined by

Show Title Display the title next to the scale.
Show Grid Lines and Scale Draw the grid lines and the scale numbers if selected.
Show Tic Marks  

Grid line location and coloring is set by

  Auto minmax Check this to have the grid lines automatically located for good readability.
Major   Draw the major grid lines (those with scale numbers)
  Distance Spacing between major grid lines
  Color major Grid line color
  Line Style Select solid, dashed, dotted major grid lines
  Line Weight Select thicker or thinner major grid lines
Minor   Draw the minor grid lines (drawn as lines, only)
  Distance Spacing between minor grid lines
  Color minor Grid line color
  Line Style Select solid, dashed, dotted minor grid lines
  Line Weight Select thicker or thinner minor grid lines

The chart fonts are selected by:

Title The axis title font.
Scale The axis scale font.

Note that axis fonts are general. They are used for all axes and for all charts. These buttons are just a quicker way to select the fonts. They can also be set in the Options Preferences dialog Charts tab.
