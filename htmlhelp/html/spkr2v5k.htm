Chamber tab (Enclosure Properties dialog)

This page contains settings for the chambers (internal volumes) of the enclosure. The enclosure can have one or two chambers in Speaker Workshop. Fields are as follows:

Box General

Volume The volume of the chamber. This is critical.
Leakage Q The leakage Q of the box caused by leakage through the dust cap and other physical leakage. Not critical and usually set to between 5 and 15.
Damping Q The Q of the box caused by internal damping. May be very high (near 100) for little internal damping.

Driver to Air (or chamber)

Select Pick the driver from a selection dialog.
None Click this to not use a driver in this chamber.
Series Resistance Enter any series resistance in the driver connection. Include resistance due to series inductors.
Number of Drivers The number of drivers used in this chamber.
Serial/Parallel If using more than one driver, enter the configuration (serial or parallel) here.

Port

Use Port Check this to use a port in this chamber.
Port is Rectangular Check this to use a rectangular (vs circular) port.
Length Enter the port length.
Port Q The port Q is non-critical and may be set to 100 for most ports.
Number of Ports The number of ports being used.
Radius The radius of the port (if circular)
Height The height of the port (if rectangular)
Width The width of the port (if rectangular)

And, in Chamber 2 only

Use this chamber In the Chamber 2 tab, select this to evaluate a two-chamber enclosure.
