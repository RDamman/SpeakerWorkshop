Record command (Sound menu, for Signals)

Use this command to record data while playing a signal through the sound board. This command brings up the Record dialog, to set things such as play and record time, channels, and volume.

The type of signal and the sampling rate are selected in the Signal Properties dialog

The recorded time data is stored as mysignal.in.l and mysignal.in.r.

To simplify use, the Edit menu Show options become available to display results quickly. Alternately, the input datasets can be viewed in another window.
