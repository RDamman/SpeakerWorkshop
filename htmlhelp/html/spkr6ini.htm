Impedance compensation dialog (Network , Insert , Impedance Compensation)

Use this dialog to add impedance compensation (often called a Zobel circuit) to the current network. The impedance compensation can be based on any driver in the network, or on the input impedance of the network. It can compensate for the inductive rise of a driver or for the resonant peak of a driver.

Fields:    
  Input Select either the current input impedance, or a specific driver.
  Compensation Select inductive rise or resonant peak to be compensated. The chosen frequency will be the frequency where the program tries to achieve a flat resistive response.

This command changes the current network, by adding an impedance compensation circuit.

In general, a driver will be selected. To use the “input impedance” option, calculate the input impedance of the network and then add this impedance compensation. Input impedance is often selected to handle more than one driver (e.g. a D’Appolito configuration), or to add compensation to a circuit that already has one type of compensation.

Select a high frequency for inductive rise to get minimal compensation. Select a low frequency to over-compensate. In general, select the correct frequency when using resonant peak compensation.
