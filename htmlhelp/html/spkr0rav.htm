Merge Port Response command
(Calculate menu for Enclosures
Driver menu for Drivers)

Use this command to merge a port response and a nearfield response for a ported enclosure. This uses the two response sets along with the known areas of the port and the driver to add the two responses together appropriately.

Ensure that your piston area (in the driver specification) and your port area are accurate before executing this command. Also, make sure you measure the two responses (nearfield driver and nearfield port) using exactly the same volume settings – so that the two responses are comparable.

See the Merge Port Response dialog for more information.
