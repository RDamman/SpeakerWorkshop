Chart Properties command (Edit menu)

Use this command to bring up the Chart properties dialog containing the properties for the currently viewed chart. This command is not available when viewing a network.

Most the of the chart properties can be changed in this dialog. Datasets can also be added to the chart and removed from the chart.

Note: many of the changes in the chart properties dialog can also be done via mouse or keyboard shortcuts using the chart itself. For example, the ranges of the X and Y axes can be set simply by dragging a box on the chart and releasing over the desired data subset. See the Chart Editing topic.
