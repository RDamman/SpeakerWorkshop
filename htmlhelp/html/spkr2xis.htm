Acoustic tab (Options Preferences dialog)

This dialog page is used to change parameters relating to acoustic measurements. These apply to nearly all acoustic (microphone-based) commands in the (Driver) Measure and (Signal) Measure menus.

Channels

Microphone Which channel the microphone is connected to (left or right). If mic input is mono then either channel will work.
Reference Which channel the reference signal is connected to (left or right or none). Set this to none if the microphone is connected using a mono Microphone input line on your sound card. Set this to the appropriate channel if you have mic input on one channel and preamp output on the other channel.

If you use a reference signal (scaled down to be 1V max) then the amplifier reference signal in the Options / Calibrate dialog is not used. If you have no reference channel then you definitely should have an amplifier reference signal in Options / Calibrate.
