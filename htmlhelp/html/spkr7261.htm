Impedance command (Measure menu, for Signals)

Use this command to measure an impedance using the Impedance Jig. This command generates an MLS signal out. The resultant input is analyzed, the two channels (source and reference) are combined, and any channel calibration is applied.

The actual Signal used to measure this response is the System signal in the System folder, where the .in.l and .in.r datasets will be created during this measurement. Signal settings are defined in the Options Preferences dialog. This dialog also contains the resistance information for the Impedance Jig.

This command creates or replaces a dataset named mysignal.Impedance. It also changes a number of datasets in the System folder.
