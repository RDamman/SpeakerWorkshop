MLS Impulse tab (Signal Properties dialog)

This page contains settings for the MLS Impulse type of signal. An MLS impulse is a pseudo-random white noise signal. The method of generating it guarantees that it can be processed in a way similar to an impulse response, but with greater noise immunity. Fields are:

Sample Size Select a value between 1024 and 128K as the number of samples to be taken. A common value is 8K or 16K. Lower values will run much quicker at lower precision.
Preemphasis Select this to have the MLS noise source be pink noise rather than white noise. This reduces accuracy at high frequencies but may improve the noise performance at lower frequencies.
Perform Correlation Leave this on.
