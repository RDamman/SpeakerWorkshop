L-Pad dialog (Network, Insert,. L-Pad)

Use this command to add an L-Pad to the current network. An L-Pad is a two-resistor network that reduces the level to a driver without changing the input impedance (assuming the driver is resistive). The L-Pad values are based on how much to decrease the driver level, and an impedance (either a resistance or an impedance based on a dataset and frequency range).

Fields:    
  Decrease Level by Enter an amount (in dB) to decrease the level by. A value of 6dB will halve the sound output, for example.
  Start If using a dataset or driver enter the start frequency to scan.
  End If using a dataset or driver enter the end frequency to scan.
  Resistance Enter the resistance the Pad is produced for.
  Driver Enter the driver whose impedance is scanned from Start to End to get the pad matching impedance.
  Dataset Enter the dataset whose impedance is scanned from Start to End to get the pad matching impedance. Usually this is a created impedance from a smaller network.

This command changes the current network, by adding an L-Pad.
