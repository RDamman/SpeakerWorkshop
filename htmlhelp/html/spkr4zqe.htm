Paste command (Edit menu)

Use this command to insert a copy of the clipboard contents at the insertion point. This command is unavailable if the clipboard is empty.

This command is not available with certain resource types. You can cut and paste parts of a network and sets of resources in the document.

To paste a resource, select the desired folder in the tree and execute Edit / Paste. This will paste the resource into that folder.

Shortcuts

Toolbar:

Keys:CTRL+V
