Sealed dialog (Calculate Sealed for Enclosures)

Use this dialog to select an optimum sealed box enclosure for a specific driver.

Fields    
  Alignment Use this to select one of the standard alignments (Custom implies not one of the standards).
  Total System Q The Q of the enclosure / driver system. This is set one a standard alignment is picked. This can also be edited to select an system Q.
  Series Resistance Enter any known resistance in the crossover as well as expected cable resistance.
  Number of Drivers If there is more than one driver in the enclosure, enter the number of drivers.
  Results This displays the results of selecting an Alignment/Q/Resistance. The box volume is for all drivers together if you have selected more than 1 driver.
Commands    
  Compare Press this command button to compare the box based on the dialog entries to the current box. See below.

This command requires that a valid driver be entered into the enclosure (see Enclosure Properties) and the driver has valid T/S parameters. Get Speaker Workshop to estimate the driver T/S parameters (via impedance measurements) or enter them manually prior to executing this command.

This command changes the current properties of the enclosure, to match the settings selected in the Sealed dialog.

The Compare Command

When you press the Compare button, Speaker Workshop builds an enclosure named “Compare” in the System directory, generates the desired curves for the compare enclosure (based on the menu settings) and charts the Compare frequency response vs the current frequency response on-screen.

Note that the Compare enclosure and associated datasets are not deleted when you exit this dialog! This allows you to create comparison charts for other than frequency response (using the Compare.xxx datasets and the current datasets) and then using the Sealed dialog you can compare impedance, cone excursion, and group delay.

For example, after running the Sealed dialog once, open the myenclosure.Group Delay dataset, Add the System\Compare.Group Delay dataset and then bring up the Sealed dialog again (while keeping the group delay dataset visible on screen). As you press compare you will note that the secondary chart (group delay) also changes each time the command is executed.
