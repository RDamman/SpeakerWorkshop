Specifics tab (Dataset Properties dialog)

This page contains:

Unit of Measure Time, Frequency, Impedance, or Other
Number of Points The number of data points in this dataset
Start The starting time / frequency / …
End The ending time / frequency / …
Amplitude Measure The amplitude is stored as absolute or dB, and as volts or power

Note that you can change these settings in this dialog and the dataset will change. Change the start frequency or end frequency to expand or reduce the range of the dataset. Change the number of points to resample the dataset to more or fewer points. Change unit of measure and amplitude measure to change the internal representation of the dataset.

Warning: dB and dBW (the two dB-based amplitude measures) have much more dynamic range than Watts and Volts (the other two amplitude measures). Switch from a dB measure at your own risk.
