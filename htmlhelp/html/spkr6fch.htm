Components tab (Network Properties dialog)

This page contains settings for the non-driver components (R,L,C, source) of a network. Select a component in the list to edit the component's properties or use Add/remove to change the component list. Fields are:

Component List

This is a list of components in the network. Click

Remove Remove a selected component from the network.
Add Add a new component to the network - automatically adds a new item, that can then be edited.

Each component has a number of editable properties. They are:

Name The name of the component.
Type The type of component (R,L,C)
Value The value of the component. Resistance for R, capacitance for C, inductance for L, and voltage for sources.
DCR Enter the DC resistance for an inductor in this field. If you leave it at 0 Speaker Workshop will assume a perfect inductor (no resistance).
