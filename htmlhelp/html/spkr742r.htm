Splice command (Calculate menu for Datasets)

Use this command to splice two datasets together. The source (A) dataset provides one part of the result and the selected (B) dataset provides the other part. The two datasets are spliced at a frequency selected in the Splice dialog.

This command is often used to splice the nearfield speaker response to the on-axis or off-axis response to get a full-frequency response approximation.

This command creates a new dataset with a Splice suffix.

For more information see the Splice dialog.
