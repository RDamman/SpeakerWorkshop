Filter dialog (Transform Filter for Datasets)

Use this command to filter data using a lowpass, highpass, or bandpass filter. Frequency data is filtered by multiplying the data by the filter response curve. Time data is filtered by executing a digital filter.

Fields    
  Type Select lowpass, highpass, bandpass, or bandstop.
  Shape Shapes include Chebychev (minimize passband ripple or minimize stopband ripple), Butterworth, and Bessel,
  Order Select an order for the filter (the filter slope is 6dB * Order)
  Attenuation For the two chebychev responses, enter the target attenuation in the stopband.
  Frequencies Look at the drawing in the dialog to determine the interpretation of 1,2,3,4 for the frequencies. For example, a lowpass butterworth will only use frequency 1 (the cutoff frequency).

This command is most commonly used to remove noise, to bandlimit data, or to simulate an active filter operation.

This command changes the current dataset.
