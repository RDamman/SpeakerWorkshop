Insert Inductor dialog (Network, Insert , Inductor for Networks)

Use this dialog to add a inductor to the current network. The inductor is not connected to anything.

Fields:    
  Name Enter any name to give the component.
  Value Enter a value for the component. Suffixes are allowed (e.g. 12K == 12000).

This command changes the current network, by adding a inductor.
