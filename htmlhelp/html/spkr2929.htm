Insert Component dialog (Network, Insert)

Use this dialog to add a passive component (R,L, or C) to the current network. The component is not connected to anything.

Fields:    
  Name Enter any name to give the component.
  Value Enter a value for the component. Suffixes are allowed (e.g. 12K == 12000).
  DC Resistance For an inductor, enter the DC resistance here. You can leave this at 0 for a “perfect” inductor.

This command changes the current network, by adding a component.
