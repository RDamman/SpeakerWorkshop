Impedance Dialog (Calculate Impedance for Datasets)

Use this dialog to calculate an impedance dataset from a set of frequency data. This dialog performs the same calculations as Measure Impedance, but after the fact for Post Mortem analysis. It uses two datasets (input data and reference data) and any channel calibration data to calculate a final response.

Impedance Jig settings such as reference resistor are found in the Option Preferences Dialog

Fields    
  Reference Response Select the reference response (the input base data). This field is required.

Actions    
  ? Click this to browse for a reference response.
