Perturb Components command (Network menu, for Networks)

Use this command to perform multiple response calculations for the current network while varying a component. The response is based on the impedances of each passive component as well as the measured impedance (Impedance) and response (On Axis) of each driver.

For more information see the Perturb Components dialog.

This command is often used to optimize a component within a network.

This command creates or replaces datasets named mynetwork.Freq.value.
