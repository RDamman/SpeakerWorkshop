Locations command (View menu)

Use this command to display and hide the Locations window, which displays information for charts.

As the mouse is moved over a chart, the Locations window shows the current Frequency (or Time) as well as the amplitude and phase for each dataset being displayed on the chart.

The two columns labeled diff1 and diff2 are the amplitude difference between the current spot and the marked spots (those spots indicated by Marker1 and Marker2).

A check mark appears next to the menu item when the Locations window is displayed.

This also has a button on the button bar to turn the window on or off.
