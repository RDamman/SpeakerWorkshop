Vu Meter command (View menu)

Use this command to display and hide the Vu Meter window, which displays information about input data values.

Every time a recording takes place (including during Measurements) the windows in this dialog are filled with the maximum, minimum, and average absolute values of the data read.

Use this to set your volume control to make the input data the right absolute value. You should try to get Minimum and Maximum near 16000 (out of a maximum of 32000).

A check mark appears next to the menu item when the Vu Meter window is displayed.

This also has a button on the button bar to turn the window on or off.
