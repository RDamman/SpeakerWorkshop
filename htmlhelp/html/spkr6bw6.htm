Impedance tab (Options Preferences dialog)

This dialog page is used to change parameters relating to impedance measurements. These parameters apply to all impedance measurements. They define the impedance jig and the Sound Board.

See Impedance Settings for a schematic.

Reference Resistor The reference resistor is usually about 10 Ohms. It should approximate the resistances you plan to measure. Large values (> 100 Ohms) may have low accuracy with most sound boards. Include any interconnect resistance in this value.
Series Resistance Input resistance seen by the sound board when measuring a straight wire. Generally this is on the order of 0.2 Ohms with alligator clips.
Test Press the Test button to measure the reference resistor and the series resistance using known resistors. This runs a wizard.
Resistance The resistance at the sound board terminals. Read the sound board specs or guess at 10K resistance. The value only matters when measuring large impedances.
Capacitance The parallel capacitance at the sound board input. A value of 0 will work for most applications. Otherwise, test it.
Test Press the Test button to measure the actual input impedance of your sound card. This runs a wizard.
