Scale dialog (Transform Scale for Datasets)

Use this dialog to scale the amplitude of a dataset.

This command is most commonly used to equalize for different input volumes when splicing data, or to normalize data viewed on a chart.

Fields    
  Method Select Add, Subtract, Multiply or Divide as methods for applying a constant.
  Scale Factor Enter the scale factor either in absolute terms (e.g. data + 4) or in dB (e.g. data increased by 4dB).

This command changes the current dataset.

Note that if the scale factor is in dB then add/subtract are actually multiply/divide.
