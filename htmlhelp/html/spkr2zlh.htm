Loop Play command (Sound menu, for Signals)

Use this command to play a signal repeatedly through the sound board. This command does not bring up the Play dialog, but it does use all settings from the previous Play command.

The type of signal and the sampling rate are selected in the Signal Properties dialog

This command is often used with a noise signal and an SPL meter to calibrate speakers for home theatre. The same thing can also be done with the microphone using Loop Record.

To stop the playing process select Sound Stop Everything.
