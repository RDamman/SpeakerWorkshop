Calibrate command (Options menu)

Use this command to select (or create) calibration curves for the system. There are three calibration curves: the microphone calibration, the channel calibration, and the amplifier reference. All are optional, although a microphone will always require a calibration for accurate results.

This command brings up the Calibrate dialog.

Calibrations are linked to the current document (because calibration curves are members of the document), thus this command should be run at least once per document to calibrate the system.
