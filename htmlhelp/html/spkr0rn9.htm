Delay command (Transform menu for Datasets)

Use this command to add or subtract a time delay from a frequency response dataset. This command brings up the Delay dialog to select how much time delay to apply (or to automatically attempt to reduce the total delay).

This command is often used to compensate for time delay phase distortion when making acoustic measurements. By setting the delay equal to the microphone delay the phase curve becomes smoother and more accurate.

This command changes the current dataset.
