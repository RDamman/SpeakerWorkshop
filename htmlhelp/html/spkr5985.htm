Intermodulation Distortion Dialog (Measure, Intermodulation Distortion, for Signals)

Use this command to measure the intermodulation distortion of a circuit or a speaker. This command generates different output signals depending on which distortion option is selected in the dialog.

Fields:

Style    
Ranges    
Equates    
Options    

This command creates or replaces a dataset named mysignal.Intermod. It may also change a number of datasets in the System folder.
