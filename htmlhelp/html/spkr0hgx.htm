Data Sets tab (Chart Properties dialog)

This page contains the list of datasets being charted.

Data Sets

This is a list of datasets being charted. Click

Remove Remove a selected dataset from the chart list.
Add Add a new dataset to the chart list - brings up the Select Dataset dialog.

In addition, for the dataset currently selected, this page lets you select

Data   Check this on to chart the amplitude data
  Color Select the line color to be charted for this dataset
  Line Style Select solid, dotted, dashed as a line style for the chart
  Line Weight Pick thin or thick lines to be used for the chart
Phase   Check this on to chart phase for the selected dataset
  Color Select the line color to be charted for this dataset's phase
  Line Style Select solid, dotted, dashed as a line style for the chart
  Line Weight Pick thin or thick lines to be used for the chart

Note that double-clicking a dataset on-screen brings up this tab with the correct dataset highlighted.

Also, click / delete of a dataset is equivalent to selecting Remove for a highlighted dataset.
