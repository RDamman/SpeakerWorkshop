Loop Record command (Sound menu, for Signals)

Use this command to record data continuously while playing a signal through the sound board. This command does not bring up the record dialog, but it does use the settings from a previous record command.

The type of signal and the sampling rate are selected in the Signal Properties dialog

The record time data is stored as mysignal.in.l and mysignal.in.r.

To simplify use, the Edit menu Show options become available to display results quickly. Alternately, the input datasets can be viewed in another window.

Set the Show option to an input channel to use the program as an audio grade oscilloscope. This command is also often used to calibrate volume settings for the sound board.

To stop the recording process select Sound Stop Everything.
