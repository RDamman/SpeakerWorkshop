IFFT dialog (Calculate IFFT for Datasets)

The IFFT dialog selects options when performing an inverse fast fourier transform (IFFT). Select the frequency range with this dialog.

Fields    
  Frequency Range Select a frequency range to be transformed. Restricting the range can improve the transform if the data is poor on the edges of the range.

Click OK to perform the IFFT transform. This creates a new dataset with an Ifft suffix.
