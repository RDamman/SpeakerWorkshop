Insert Driver dialog (Network, Insert , Driver for Networks)

Use this dialog to add a driver to the current network. The driver is not connected to anything.

Fields:    
  Name Enter any name to give the driver in the network.
  Offset Enter a value for the driver offset. This is the distance the driver is offset from a vertical plane.
  Individual Response Check this on to have the driver’s individual response calculated when the network response is calculated.
  Flip Polarity Check this on to have the driver connected in reverse polarity. Off is normal polarity.

This command changes the current network, by adding a driver.
