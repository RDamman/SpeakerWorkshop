Sound Card Calibration

Calibrating your sound card is something that you should always do at least once. The Speaker Workshop automatic calibration routines are quite accurate and at worst may be useful as a starting point.

There are 3 different calibration tests:

Channel Difference This uses a loop cable to find the differences between the left and right input channels.
Sound Card Source Impedance This measures the input resistance and capacitance of your sound card.
Impedance Jig This test uses calibration resistors to find the actual resistances for your impedance jig.

Note that a single impedance jig with 3 reference resistors (short, 10 ohms, and 10K ohms) is sufficient to handle all three calibration sequences. The shorted jig suffices as a loop cable in the channel difference test. The jig with a 10K reference can be used for the source impedance test. The impedance jig test uses the normal 10 ohm (or so) reference.

Step by Step Calibration

1.Channel difference - select the Options Calibrate command and press the Test button under channel calibration. The directions will call for a loop cable plugged into the sound card input and output.

1.Sound Card Source Impedance - execute the Options Preferences command and select the Impedance tab. Press the Test button in the Sound Card Source Impedance group. The instructions will call for using a cable with a 10K series resistor. You can also use an impedance jig with a 10K reference.

1.Impedance Jig - execute the Options Preferences command and select the Impedance tab. Press the Test button in the Impedance Jig group. The instructions call for setting up the impedance jig as in normal use. The sound card is calibrated by examining 2 or more known resistors and then estimating the best fit reference resistance and series resistance. Even if you think you know your impedance jig values try this at least once.
