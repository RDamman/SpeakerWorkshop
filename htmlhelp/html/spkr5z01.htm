The Impedance Jig is a cable, a resistor, and a few connectors that Speaker Workshop uses to measure impedance of drivers and components. Set the jig parameters in Options Preferences (Impedance tab). The jig looks like:

For an interesting overview, see Cables and Connectors

The capacitance value entered into Options Preferences should be the capacitance of the branches from the resistor to the input miniplug. If you don’t know this, a value of 10pF will work fine.

The input resistance should be the input impedance of the sound card. This is often in the user’s manual for the sound card. You can measure the input resistance and capacitance by pressing the Test button in the Options Preferences Impedance tab.
