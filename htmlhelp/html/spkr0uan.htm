Splice Dialog (Calculate Splice for Datasets)

Use this dialog to splice two datasets together. The source (A) dataset provides one part of the result and the selected (B) dataset provides the other part.

This command is often used to splice the nearfield speaker response to the on-axis or off-axis response to get a full-frequency response approximation.

Fields:    
  A The A source dataset (the dataset currently being viewed). Not editable.
  B The selected B dataset. Editable.
  Frequency Select the frequency at which the splice takes place.
  Methods Either A or B can be the high frequency portion of the result. Also, the frequency resolution can be based on A or B data. Check the appropriate boxes to use B rather than A (the default).

Select OK to perform the splice. This command creates a new dataset with a Splice suffix.
