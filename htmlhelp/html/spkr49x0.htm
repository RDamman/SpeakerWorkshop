Edit menu commands

The Edit menu offers the following commands:

Undo Reverses previous editing operation.
Redo Redoes previously undone editing operation.
Cut Deletes data from the document and moves it to the clipboard.
Copy Copies data from the document to the clipboard.
Paste Pastes data from the clipboard into the document.
Delete Delete the selected object or area.
Make Chart Default Make the current chart format the default for all new charts with this unit of measure.
Use Chart Default Use the current default chart format with this chart.
Properties Brings up a dialog specific to the currently selected resource.
Chart Properties Brings up a dialog to change the chart properties for the currently viewed chart.
