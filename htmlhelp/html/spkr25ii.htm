Chart tab (Options Preferences dialog)

This dialog page is used to change the general appearance of charts.

The font selections apply to all charts - resources of chart type as well as the chart display of datasets. Press the ? button next to a font to browse for a new font.

The color selections also apply to all charts. You can change the background color, the default grid color, and the interior grid color. Click the colored button to get a color selection dialog that lets you pick a different color for your charts.
