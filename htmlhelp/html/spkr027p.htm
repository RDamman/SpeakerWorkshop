Truncate dialog (Transform Truncate for Datasets)

Use this dialog to reduce the frequency (or time) range of a dataset.

Fields    
  Use Markers Use the current marker settings as the range limits
  Use Range Enter a low/high range to truncate the data to.

This command is most commonly used to remove noisy or invalid data from the frequency extremes of a dataset.

This command changes the current dataset, reducing the number of data points.
