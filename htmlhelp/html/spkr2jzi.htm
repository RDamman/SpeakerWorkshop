Networks tab (Options Preferences dialog)

This dialog page is used to change the general appearance of networks. These parameters apply to all networks.

Label Font The font used to display component names in the network.
Value Font The font used to display the value for components in the network.
Display Precision This determines how values are displayed in the network. Select one of the predefined formats. All network components being displayed use this format.
Draw resistors as Draw resistors as boxes or squiggles. This toggles between the two standard resistor drawing methods.
Draw component endpoints Draw small circles to show component ends (or not).
