Sealed command (Calculate menu for Enclosures)

Use this command to select an optimum sealed box enclosure for a specific driver. See the Sealed dialog for more information.

This command requires valid T/S parameters for the driver. Get Speaker Workshop to estimate the driver T/S parameters (via impedance measurements) or enter them manually prior to executing this command.

This command changes the current properties of the enclosure, to match the settings selected in the Sealed dialog.
