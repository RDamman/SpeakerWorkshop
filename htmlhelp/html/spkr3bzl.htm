Impedance compensation command (Network menu, Insert submenu, for Networks)

Use this command to add impedance compensation to the current network. The impedance compensation can be based on any driver in the network, or on the input impedance of the network. It can compensate for the inductive rise of a driver or for the resonant peak of a driver.

See the Impedance Compensation dialog for more information.

This command changes the current network, by adding an impedance compensation circuit.
