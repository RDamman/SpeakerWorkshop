Estimate Parameters command (Driver menu for Drivers)

Use this command to estimate the T/S parameters for a driver based on measured results. This command brings up the Estimate Parameters dialog, which selects a frequency range to be used during the estimation process. For more information, see the Estimate Parameters Dialog discussion.

This command is a necessary part of testing and prototyping a driver/enclosure combination. It is required for any driver that will require an enclosure. Generally, the Measure Impedance in Free Air and Measure Impedance in Sealed Box commands are executed prior to this command.

This command changes the current properties of the driver, inserting new T/S parameters based on input data.
