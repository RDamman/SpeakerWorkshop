Import and Export Formats

Speaker Workshop can export to and import from a number of different formats. These are:

Impedance data ZMA The IMP ascii impedance format.
Frequency data FRD The IMP ascii frequency format.
Time data IMP The IMP ascii time format.
Signals WAV Windows standard wave files.
Network CIR An ascii format similar to SPICE format.

Note that signals are imported by being defined as custom, not through the Resource Import command.

The formats are:

Impedance (ZMA)

Line Name Description
1…N Data Values Frequency, Amplitude, and Phase in floating point

Frequency (FRD)

Line Name Description
1…N Data Values Frequency, Amplitude, and Phase in floating point

Time (IMP)

Line Name Description
1 Sample Size Number of data points (integer)
2 Sample Size Number of data points (integer)
3 Sample Rate Number of samples per second (integer)
4 ??? unknown - set to 1
5 Date Sample date and time in ascii hh:mm:ss mm/dd/yy
6 Starting Marker An integer - used by IMP for gating.
7 End Marker An integer - used by IMP for gating.
8 ??? unknown, set to 1.0000000E+00
9…N Data Values Frequency, Amplitude, and Phase in floating point

Wave (WAV)

See the relevant Microsoft documentation about WAV files. Speaker Workshop WAV files are compatible with all Windows 95 wav programs.

Circuit (CIR)

Line Name Description
1…N Data Values This data is formatted one line per component, using the fields below. For example

R_R1 $N_2 $N_3 1.00K 45 50 0
Fields: 1 R_name if a resistor,

C_name if a capacitor,

L_name if an inductor,

V_name if a voltage source,

D_name if a driver
  2 $N_node1
  3 $N_node2
  4 value
  5 X Position
  6 Y Position
  7 Orientation
