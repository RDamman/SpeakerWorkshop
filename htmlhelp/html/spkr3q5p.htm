Calculation menu commands Enclosure menus

The Calculation menu offers the following switches and commands:

Switches

Cone excursion Check to calculate cone excursion when parameters change.
Frequency response Check to calculate frequency response when parameters change.
Group Delay Check to calculate group delay when parameters change.
Impedance Check to calculate impedance when parameters change.
Transient response Check to calculate transient response when parameters change.

Commands

Sealed... Tune a sealed enclosure to the desired response.
Vented... Tune a vented enclosure to the desired response
Merge Port Response... To merge a port and driver nearfield responses appropriately for a vented (ported) enclosure.
