Set Equivalent dialog

This dialog lets you set the electrical equivalent circuit of a driver manually, after trying the automatic approach and being possibly unhappy with the fit.

Press the Chart button to see the impedance chart with the new settings. This uses the mydriver.Estimate chart so make sure you have tried to estimate parameters at least once (which creates that chart).

This works best if you have the driver window on-screen along with the mydriver.Free Air and mydriver.Estimate datasets in the same chart. Then, when you press Chart you can compare the free air reading with the estimate.

Estimated T/S parameter Description

QMS Mechanical Q
QES Electrical Q
QTS Q of the total system
FS Resonant frequency in free air
