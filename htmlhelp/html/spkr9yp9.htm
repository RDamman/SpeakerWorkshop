Delay Dialog (Transform Delay for Datasets)

Use this dialog to add or subtract a time delay from a frequency response dataset.

This command is often used to compensate for time delay phase distortion when making acoustic measurements. By setting the delay equal to the microphone delay the phase curve becomes smoother and more accurate.

Fields    
  Remove Excess Delay Select this to remove the excess time delay in acoustic measurements
  Delay Time Enter a time in msec to delay the data by.

Press OK to transform the change the current dataset.

Press Apply to preview the changes to the dataset.
