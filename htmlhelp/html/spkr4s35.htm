Time Response command (Measure menu, for Signals)

Use this command to measure the time (pulse) response of a circuit. This command generates an MLS signal out. The result is the source input analyzed. This analysis does not use the reference input.

The actual Signal used to measure this response is the System signal in the System folder, where the .in.l and .in.r datasets will be created during this measurement. Signal settings are defined in the Options Preferences dialog.

This command creates or replaces a dataset named mysignal.Time. It also changes a number of datasets in the System folder.
