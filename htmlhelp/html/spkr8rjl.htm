Insert Resistor dialog (Network, Insert , Resistor for Networks)

Use this dialog to add a resistor to the current network. The resistor is not connected to anything.

Fields:    
  Name Enter any name to give the component.
  Value Enter a value for the component. Suffixes are allowed (e.g. 12K == 12000).

This command changes the current network, by adding a resistor.
