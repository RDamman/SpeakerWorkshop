Combine dialog (Calculate Combine for Datasets)

Use this dialog to combine two datasets arithmetically.

Fields:    
  Execute The top dataset is the currently viewed dataset. The second dataset can be selected via the browse (?) button.
  Operation Select Add, Subtract, Multiply, or Divide.

Actions:    
  ? This brings up the select object dialog, to select the second dataset.
  OK Click OK to perform the arithmetic.

The source dataset is changed by this command.

Example:

mysignal.in.l divide mysignal.in.r produces left input /right input as mysignal.in.l.combine.
