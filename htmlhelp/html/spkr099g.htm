Drivers tab (Network Properties dialog)

This page contains settings for the drivers in the network. Networks can contain more than 1 driver (and more than 1 of each kind of driver). Select a driver in the driver list to edit the driver's properties, or use Add/Remove to change the driver complement.

Driver List

This is a list of drivers in the network. Click

Remove Remove a selected driver from the network.
Add new Add a new driver to the network - brings up the Select a Driver dialog..

Each driver in the network has a number of editable properties. They are:

Name The display name of the driver. This can be different from the actual name of the driver.
Offset The linear offset of the driver. This is relative to other drivers in the network and allows for sloped cabinets and physical driver offsets.
Flip Polarity Select this to simulate connecting the driver flipped (plus to ground, minus to signal). For crossovers where one driver may be connected out of phase.
Individual response Check this to have Speaker Workshop calculate the driver's individual response when calculating the total network response. This is used with the automatic overview chart feature.

In addition, each driver's location and connections can be edited via:

Nodes Which nodes this driver is connected to
Position X,Y location of the driver

Node numbers are constantly recalculated, and are only valid during this dialog. Connections based on the node numbers are sticky, however. Position information is based on a maximum size of 1024x1024 for the network.

Obviously node numbers and locations are more easily set with the mouse in the network display using drag and drop. See editing networks with the mouse.
