Measurements tab (Options Preferences dialog)

This dialog page is used to change parameters relating to measurements. These apply to nearly all commands in the (Driver) Measure and (Signal) Measure menus.

Sample Rate Use low sample rates (11 KHz) for low frequency testing, such as woofer impedance and nearfield response. Use high sample rates (44 KHz) for general frequency response testing.
Sample Size Increasing the size increases the resolution (smaller step sizes) and improves noise immunity at the cost of more data. Generally 8K - 16K are reasonable sample sizes for a medium speed Pentium.
Volume Set this to 0…100 to get volume changed logarithmically proportional to the #. This is via the Windows mixer and is usually an analog volume control.
Reverse Channels Switch source and reference input channels - for times when the leads are backwards.
Use Preemphasis This applies a low pass filter to the MLS data. The volume can be increased without overload, increasing the low frequency accuracy.
Precision Results This output field shows the range and resolution of the measurement using the selected Sample Rate and Sample Size.
Repeat Count Set this to a value greater than 1 to have the MLS data repeated multiple times. The program will average the readings and this will reduce noise in the sampling (theoretically by 6dB for each doubling of the count).

For acoustic measurements make sure the measurement is long enough to let echoes die away if you use repeat counts > 1. Even then, this may not be recommended. For impedance measurements this works extremely well – try a count of 6.
