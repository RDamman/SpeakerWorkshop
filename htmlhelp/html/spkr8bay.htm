General tab (Chart Properties dialog)

This page contains:

Title The title of the chart (printed on the top of the chart)
Font Select a title font (this is also settable in the Options Preferences Chart tab).

Description Enter a description. Descriptions can be more than one line: use Ctrl-Enter to add new lines.
Created When this Chart was created (output field).
Last Edited When the Chart was last edited (output field)

Show Legend Display the legend on the chart.
