Passive Calculator dialog (Options Calculator Passive)

Use this dialog to calculate impedances for passive components at specific frequencies.

Fields    
  Input Type Select the resistor, capacitor, or inductor radio button to pick a type.
  Value Enter the value of the component.
  Frequency Enter the frequency of interest.

Results  
  The impedance is shown as real and imaginary (rectangular) as well as amplitude and phase (polar).

Shortcuts

None
