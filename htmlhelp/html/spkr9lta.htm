Data tab (Driver Properties dialog)

This page primarily contains references to other datasets that together define the driver.

Impedance The dataset that defines the impedance of the driver as a function of frequency. Used for network calculations.
Pulse Response The pulse response of the driver. Not used by Speaker Workshop.
Frequency Response The on-axis frequency response of the driver. Used for network calculations.
Nearfield Response The response of the driver in the nearfield. Not used by Speaker Workshop.
Response at 30 deg The response of the driver 30 degrees off-axis. Not used by Speaker Workshop.
Response at 60 deg The response of the driver 60 degrees off-axis. Not used by Speaker Workshop.

These fields are only used for T/S parameter estimation.

Impedance in Free Air The measured impedance of the driver in free air.
Use this DC Res The DC resistance of the driver, if known. If this is not checked, then the DC resistance will be estimated with the other T/S parameters. If this is checked, the DC resistance will be as entered.
Impedance in Sealed Box The measured impedance of the driver in a closed box.
Sealed Volume The volume of the closed box for T/S parameter estimation.
Impedance with Added Mass The measured impedance of the driver when mass is added to the cone
Added Mass The amount of mass added to the cone.

To change a dataset linkage, click the ? button next to the dataset name. This brings up the usual Select Dataset dialog where a dataset can be picked. To remove a dataset linkage just click the X button next to the dataset name.

To select between the added mass and the delta compliance (sealed box) methods, just click the radio button on the left of the appropriate data set. Only one of these needs to be filled in to estimate driver parameters (you don’t use both added mass and sealed box methods).

The units of measure for volume and mass are selectable, click the unit to switch between the various metric and English units.
