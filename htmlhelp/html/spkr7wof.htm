Close command (File menu)

Use this command to close all windows containing the active document. Speaker Workshop suggests that you save changes to your document before you close it. If you close a document without saving, you lose all changes made since the last time you saved it. Before closing an untitled document, Speaker Workshop displays the Save As dialog box and suggests that you name and save the document.
