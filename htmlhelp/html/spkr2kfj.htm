Calculate Response command (Network menu, for Networks)

Use this command to perform the response calculation for the current network. The response is based on the impedance of each passive component as well as the measured impedance (Impedance) and response (On Axis) of each driver.

This command will also calculate input impedance of the network - if so selected in the network properties dialog. Other options include generating individual frequency response results for each driver and creating an overview chart with all responses displayed. All of these options are in Network Properties (and get saved with the Document).

This command brings up the Calculate Response dialog. Pressing OK creates or replaces datasets named mynetwork.Total Freq and mynetwork.Impedance. It may also create or replace mydriver.Freq datasets for each driver.
