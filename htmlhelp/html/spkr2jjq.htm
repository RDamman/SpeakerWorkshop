Stock Crossover dialog (Network, Insert,. Stock Crossover)

Use this command to add a standard crossover to the current network. The crossover is based on a selected resonant frequency and resistance.

Fields:    
  Order Select 1st, 2nd, 3rd, or 4th order crossovers.
  Style Select highpass (tweeter) or lowpass (woofer).
  Type The types available are dependent on the order of the crossover. E.g. a 1st order crossover must be butterworth. Second order can pick from Bessel, Butterworth, Chebychev, and Linkwitz-Riley.
  Resonant frequency Select the resonant frequency.
  Target Resistance Select an approximate resistance for the crossover to match to.
  Driver Select a driver to match to. This will match the crossover to the driver’s impedance at the selected frequency.
  Dataset Pick an impedance dataset to match to. This will match the crossover to the dataset’s impedance value at the selected frequency. Usually this will be a calculated network impedance so you can match a crossover to the in-process network.

This command changes the current network, by adding a stock crossover circuit.
