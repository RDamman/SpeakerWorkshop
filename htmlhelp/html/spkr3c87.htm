Create Goal dialog (Network, Create Goal…)

Use this command to build a goal dataset. This dataset simulates a perfect driver with a given crossover type. The crossover frequency is based on a selected resonant frequency. The crossover base value is based on another dataset or driver or a given value.

Fields:    
  Order Select 1st, 2nd, 3rd, or 4th order crossovers.
  Style Select highpass (tweeter) or lowpass (woofer).
  Type The types available are dependent on the order of the crossover. E.g. a 1st order crossover must be Butterworth. Second order can pick from Bessel, Butterworth, Chebychev, and Linkwitz-Riley.
  Resonant frequency Select the resonant frequency.
  Base Value Select a value for the base dataset.
  Driver Select a driver to match to. This will match the base value to the driver’s frequency response at the selected frequency.
  Dataset Pick a response dataset to match to. This will match the base value to the dataset’s amplitude at the selected frequency.
  Frequency Start/End Pick a start and end frequency. The selected dataset or driver is averaged over this range to calculate the base value for the goal.

This command creates a new dataset named mynetwork.Goal. If the Goal dataset already exists it is overwritten with the new goal values.
