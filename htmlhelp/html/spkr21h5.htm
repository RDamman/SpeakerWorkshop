Record Dialog (Sound, Record, for Signals)

With this dialog, select things such as play and record time, channels, and volume.

The type of signal and the sampling rate are selected in the Signal Properties dialog

The recorded time data is stored as mysignal.in.l and mysignal.in.r.

Fields:    
  No Output Check to disable output. Used when recording from CD or other non-sound board source.
  Output Volume Set the volume from 0 to 100 (logarithmically).
  Channels Select left, right, or both. The sound is always mono if both are selected.
  Input calibration Select which channel to use as the reference channel.
  Input data The input data channel.
  Play time How long to play the signal.
  Record time How long to record. This may be longer than the play time if the recording is acoustic and there are echoes to record.
  Type Select Frequency to automatically get a frequency chart based on the recorded time data. Similarly time shows the recorded time data. Impedance uses the impedance in the Options Preferences Measurements tab to calculate impedances based on the recorded data.

To let you use Speaker Workshop as a dual trace oscilloscope, the Edit menu Show options become available to display results in the signal’s view window. Alternately, the input datasets can be viewed in another window.
