Filter command (Transform menu for Datasets)

Use this command to filter data using a lowpass, highpass, or bandpass filter. Frequency data is filtered by multiplying the data by the filter response curve. Time data is filtered by executing a digital filter. This command brings up the Filter dialog to select the filter parameters such as cutoff frequency.

This command is most commonly used to remove noise, to band limit data, or to simulate an active filter operation.

This command changes the current dataset.
